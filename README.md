# MyTOP100Movies
Uma aplicação web que permite os usuários criarem uma lista com os seus cem filmes favoritos.

## Objetivo secundário

Este projeto, além de sua proposta (não original), busca ser um lugar onde posso aplicar meus conhecimentos e, assim, práticar.


***
Todos os créditos a respeito do uso dos filmes vão para o The Movie Database (themoviedb.org)
